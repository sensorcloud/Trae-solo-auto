### ⚠️ GitHub - 需要手动推送

**仓库地址**: https://github.com/ywtech/EdgeAgent-Hub

**当前状态**: SSH连接超时，需要使用HTTPS方式推送

#### 推送方式一：使用Personal Access Token

1. 在GitHub生成Personal Access Token:
   - 访问: https://github.com/settings/tokens
   - 生成新token，勾选 `repo` 权限

2. 配置Git认证:
   ```bash
   git remote set-url github https://<USERNAME>:<TOKEN>@github.com/ywtech/EdgeAgent-Hub.git
   ```

3. 推送代码:
   ```bash
   git push github master
   ```

#### 推送方式二：使用GitHub CLI

1. 安装GitHub CLI:
   ```bash
   # macOS
   brew install gh
   
   # 或访问 https://cli.github.com
   ```

2. 认证:
   ```bash
   gh auth login
   ```

3. 推送:
   ```bash
   gh repo sync ywtech/EdgeAgent-Hub
   # 或直接
   git push https://github.com/ywtech/EdgeAgent-Hub.git master
   ```

#### 推送方式三：手动下载上传

1. 在当前环境创建补丁文件:
   ```bash
   git format-patch -1 697f688 -o /tmp/patches/
   ```

2. 下载补丁文件到本地

3. 在有GitHub访问权限的环境中应用并推送
