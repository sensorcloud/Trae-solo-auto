# EdgeHub - 算电运协同产业互联网平台

## 项目简介

EdgeHub 是一个开源的**算电运协同产业互联网平台**，致力于实现能源、算力、运营三者的深度融合与智能协同。平台通过创新的"能源-算力-运营"三层商业模式，构建绿色低碳、高效智能的新型基础设施。

### 核心理念

- **能源层（L1）**：绿电生产 + 储能调度，实现清洁能源的高效利用
- **算力层（L2）**：IaaS + MaaS 服务，提供弹性可扩展的计算能力
- **运营层（L3）**：智能体服务 + IoT连接，赋能产业数字化转型

### 平台价值

| 维度 | 价值主张 |
|------|----------|
| **绿色低碳** | 通过算电协同调度，最大化绿电利用率，降低碳排放 |
| **降本增效** | 智能调度算法优化资源配置，降低运营成本 30%+ |
| **开放生态** | 开源核心，支持多租户、多云、多集群管理 |
| **安全可信** | 零信任架构，端到端加密，满足企业级安全要求 |

## 核心功能模块

### 1. 能源市场模块

能源市场模块提供完整的能源管理与交易能力：

- **电源管理**：支持光伏、风电、水电、储能等多种能源类型的接入与管理
- **储能调度**：智能充放电策略优化，支持峰谷套利、需求响应
- **能源交易**：支持现货交易、中长期合约、绿证交易
- **虚拟电厂（VPP）**：聚合分布式能源资源，参与电网调度

## 快速开始

### 环境要求

- Kubernetes 1.28+
- Go 1.21+
- Docker (可选)
- Helm 3.x (推荐)

### 安装部署

#### 方式一：使用 Helm 部署（推荐）

```bash
# 克隆项目
git clone https://github.com/edgehub/edgehub.git
cd edgehub

# 添加依赖仓库
helm repo add bitnami https://charts.bitnami.com/bitnami
helm repo add karmada https://github.com/karmada-io/karmada/releases

# 安装 EdgeHub
helm install edgehub ./charts/edgehub \
  --namespace edgehub-system \
  --create-namespace \
  --set image.tag=latest

# 验证安装
kubectl get pods -n edgehub-system
```

#### 方式二：使用 Makefile

```bash
# 克隆项目
git clone https://github.com/edgehub/edgehub.git
cd edgehub

# 构建二进制文件
make build

# 构建镜像
make docker-build

# 部署到 Kubernetes
make deploy
```

### 使用 CLI

```bash
# 安装 CLI 工具
make install-cli

# 或使用脚本安装
curl -sSL https://raw.githubusercontent.com/edgehub/edgehub/main/scripts/install-cli.sh | sh

# 登录
edge login --endpoint http://localhost:8080

# 查看节点列表
edge node list

# 提交计算任务
edge job submit --file job.yaml

# 查看算力市场
edge market list-offers

# 查看能源状态
edge energy status
```

### 本地开发

```bash
# 安装依赖
go mod download

# 运行测试
make test

# 启动开发服务器
make dev

# 代码格式化
make fmt

# 代码检查
make lint
```

```bash
make test

# 运行所有检查
make check
```

### Commit 规范

使用 [Conventional Commits](https://www.conventionalcommits.org/) 规范：

- `feat:` 新功能
- `fix:` Bug 修复
- `docs:` 文档更新
- `style:` 代码格式（不影响功能）
- `refactor:` 重构
- `test:` 测试相关
- `chore:` 构建/工具相关

### 开发环境设置

```bash
# 安装 pre-commit hooks
pre-commit install

# 安装开发依赖
go install github.com/golangci/golangci-lint/cmd/golangci-lint@latest
```
