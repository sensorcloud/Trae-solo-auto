# EdgeHub - 边缘算力集群聚合平台

## 概述

EdgeHub 是一个开源的边缘算力集群聚合平台，专注于将分散的边缘计算资源与多样化的计算需求进行智能匹配。

## 核心特性

- **异构算力管理**: 支持 CPU、GPU、NPU、FPGA 等多种计算资源的统一管理
- **智能调度**: 基于 Kueue + Volcano + OpenFuyao 的高级调度引擎
- **GPU 虚拟化**: 基于 HAMi 的 GPU 切分与共享
- **算力市场**: 实时价格聚合、智能推荐、竞价交易
- **多集群管理**: 基于 Karmada 的跨集群编排
- **性能评测**: 标准化的基准测试与节点评分系统

## 快速开始

### 环境要求

- Kubernetes 1.28+
- Go 1.21+
- Docker (可选)

### 安装

```bash
# 克隆项目
git clone https://github.com/edgehub/edgehub.git
cd edgehub

# 构建
make build

# 部署到 Kubernetes
make deploy

# 或使用 Helm
helm install edgehub ./charts/edgehub --namespace edgehub-system --create-namespace
```

### 使用 CLI

```bash
# 安装 CLI
make install-cli

# 登录
edge login

# 查看节点
edge node list

# 提交任务
edge job submit --file job.yaml

# 查看市场
edge market list-offers
```

## 主要模块

### 算力资源管理

- 节点注册与接入
- 异构算力管理 (CPU/GPU/NPU/FPGA)
- 资源池化与虚拟化
- 节点监控与状态管理

### 计算任务调度

- 智能任务调度引擎
- 多类型任务支持 (容器/AI推理/批处理/无服务器)
- 跨集群调度
- GPU 调度优化

### 算力价格对比与交易

- 实时价格聚合
- 智能价格推荐
- 算力交易市场
- 计费与结算

### 性能评测

- 基准测试框架
- AI 推理性能评测
- 节点评分系统
- 实时性能监控

## 技术栈

| 组件 | 技术 |
|------|------|
| 容器编排 | Kubernetes 1.28+ |
| 多集群管理 | Karmada |
| 批处理调度 | Kueue + Volcano |
| 高级调度 | OpenFuyao |
| GPU 虚拟化 | HAMi |
| 服务网格 | Linkerd |
| 存储 | Rook-Ceph |
| AI 推理 | vLLM |
| 分布式计算 | Ray + KubeRay |
| 监控 | Prometheus + Grafana |

## 配置

配置文件位于 `config/` 目录：

```yaml

  port: 8080
database:
  host: localhost
  port: 5432
  name: edgehub
redis:
  host: localhost
  port: 6379
scheduler:
  enabled: true
  type: kueue
```
